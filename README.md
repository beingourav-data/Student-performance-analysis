# Student Performance Analysis Dashboard (Power BI)

Interactive Power BI dashboard to analyze student performance by gender, test preparation, parental education, and ethnicity.

## Steps Done
- Imported dataset into Power BI
- Cleaned and transformed data in Power Query
- Created calculated columns: Total Marks, Score Percentage
- Built DAX measures for KPIs and analysis
- Designed interactive dashboard with slicers and charts

## Dashboard KPIs
- Student Count
- Avg Reading, Writing, Math Score
- Avg Percentage Score
- Max Score %

## Visuals Used
- KPI Cards
- Bar Charts
- Donut Chart
- Slicers (Gender, Test Prep, Ethnicity, Parental Edu)
